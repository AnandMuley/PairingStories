Rule 1 - High Card:

High card is the highest rank card a player has got.

Scenario 1 : If a player has got a high card with rank greater than that of the other players, then he wins.

Input:
P1 (John) : TEN-CLUB, JACK-HEART, JACK-DIAMOND


P2 (Bob) : NINE-CLUB, THREE-HEART, EIGHT-DIAMOND

Output:
John has won by high card: JACK – HEART

Scenario 2 : If two or more players have got a high card of same rank, then the player having high card with greater suit rank wins.

Input:
P1 (John) : TEN-CLUB, THREE-HEART, JACK-DIAMOND


P2 (Bob) : NINE-CLUB, THREE-HEART, JACK-SPADE

P2 (Phill) : THREE-CLUB, SIX-HEART, JACK-HEART

Output:
Phill has won by high card: JACK - HEART

Scenario 3 : If two or more players have a high card with same rank and suit, then next high card is compared and so on

Input:
P1 (John) : TEN-CLUB, JACK-HEART, JACK-DIAMOND, KING-HEART, ACE-SPADE


P2 (Bob) : NINE-CLUB, THREE-HEART, EIGHT-DIAMOND, KING-HEART, ACE-SPADE

Output:
John has won by high card: JACK – HEART

Scenario 4 : If the winner cannot be decided based on above rules the game is drawn

Input:
P1 (John) : TEN-CLUB, JACK-HEART, JACK-DIAMOND


P2 (Bob) : NINE-CLUB, THREE-HEART, EIGHT-DIAMOND

Output:
Match was drawn
